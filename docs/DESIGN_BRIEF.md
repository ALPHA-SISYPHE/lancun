# 设计 Brief

## 文档状态

- 版本：4.0
- 状态：已对齐根目录 [`DESIGN.md`](../DESIGN.md)（大疆摄影风·通透海水蓝，2026-07-17）

## 1. 设计目标

以海洋深蓝为全站基底，通过 **真实摄影 + 雾感海水蓝 UI + 白内容岛** 呈现通透、克制的保护叙事；参考 Convex 仅借 **地球区交互结构**，视觉以 `DESIGN.md` 为准。

## 2. 情绪路径

```text
海洋之美 → 好奇探索 → 数据与生物理解 → 保护行动 → 持续参与
```

## 3. 视觉关键词

- 大疆式通透、少即是多、浅雾蓝白岛、ghost 按钮、透明导航、宁静
- Hero 海面摄影/视频；内页正文在 **16px 圆角白岛** 内

## 4. 方向（v4）

- 首屏：电影感纪录片 + overlay 导航
- 全站底：`#0F172A → #1E3A8A` 流体渐变（可选 `ocean-bg.jpg`）
- 主色：`#3B82F6`；点缀 `#5EEAD4` 仅小面积
- **无** 全站光池/caustics 等堆装饰

## 5. 避免事项

（同 v2）儿童卡通、廉价霓虹、满屏玻璃、无意义动画、风格不一致。

## 6. 权威文档

- **实现 mirror：** `docs/DESIGN_SYSTEM.md` v4
