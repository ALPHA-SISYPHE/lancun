# 澜存 · LANCUN OCEAN PROTECTION

Web 编程期末大作业：海洋保护展示与互动课程项目。

## 目录

- `index.html`：首页。
- `pages/`：地图、数据、生物档案与行动中心的独立页面。
- `assets/css/`：通用与页面专属样式。
- `assets/js/`：前端交互脚本。
- `assets/media/`：后续正式使用的图片、视频、音频资源（本地保留，默认不入仓）。
- `docs/`：项目长期规则、需求、设计与验收文档。
- `references/`：课程要求、人工迭代笔记、视频参考和外部设计资源（本地保留，默认不入仓）。
